== Description ==

Creativ Themes starter sites importer.

== Installation ==

1. Upload `creativ-demo-importer` to the `/wp-content/plugins/` directory;
2. Activate the plugin through the 'Plugins > Installed Plugins' menu in WordPress dashboard;
3. Done!
